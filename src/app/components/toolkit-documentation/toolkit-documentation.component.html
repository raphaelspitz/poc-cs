Welcome to Toolkit IWM

The goal of this project is to standardize a UI library reusable in terms of components and sass.

We are creating this library to avoid using too many external libraries and problems of accessibility in case if it is needed to be used with a custom specific design asked to developed on our project.

Also, we are centralizing the css to match with the design guidelines so that it has a close naming convention associated to it and to prevent using inline css. Therefore we will be able to reduce the amount of css.

This project is a separate project created under the IWM folder, we did it to avoid creating conflicts with the existing one, however we have created modules that can be added inside the project and it works.

The purpose is too only standardize the code and to only add components that has be customized if it is not available in out of the box widgets or if it doesn't match with the requirements.

And as last purpose it will give the possibility to create knowledge sharing with documentations available between developers and other participants. This project is available on git and can be run with npm run proxy-demo on port 4202.

Every devs can create pull request on it if needed and participate :)

About brandings fonts, colors, animations, icons

this project has the original EP and ET fonts, a part of the colors from the design guidelines are also there, icons are available as well

borders, backgrounds, hover effects transitions are presents with css naming conventions that matches on design guidelines for examples : border-bottom-gray-3, bg-gray-2, bg-gray-1, hover-bg-gray-1
